# Multispectral Weed Detection using Deep Learning

## Overview
This repository contains implementation of classification, object detection, and segmentation models using multispectral imagery (NIR, Red Edge).

## Tasks
- Classification (MobileNetV3)
- Object Detection (YOLOv8)
- Instance Segmentation (YOLOv11)
- Semantic Segmentation (DeepLabV3+, SegFormer)

## Reproducibility
- Training scripts and preprocessing pipelines are provided
- Dataset links are included (datasets not redistributed)

## Requirements
See requirements.txt

## Usage
Run notebooks in /notebooks folder
